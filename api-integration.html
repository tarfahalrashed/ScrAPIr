- Name: The name of the parameter as sepcified in the API documentation. (e.g. in NYT Article Search API q=Trump, q is the parameter name)

6. Pagination

Pagination Information

Demo Example

When you access web APIs there’ll be a lot of results to return. For that reason, APIs paginate the results to make sure responses are easier to handle. APIs give you results in pages, every page has a default and a maximum number of results, here we need you to provide max number of results per page.

If you want to get more results than the max number of results per page allowed by the API, there are two different ways to loop through pages: (1) Some APIs use index or offset to go through multiple pages(e.g. In Wikipedia search API, they use ‘offset’ as the page parameter. offset=0 is the first page, offset=1 is the second page, etc). (2) Other APIs will provide parameters that will indicate the current page, in which you can assign it to the next page parameter to get more results (e.g. in YouTube search API, the current page parameter is ‘pageToken’ and to go to the next page we assign ‘pageToken’ to the ‘nextPageToken’).
